# Quizz Disney

Jeu de quiz multiple-choice sur le thème de Disney, avec des questions de difficulté croissante.

## Fichiers

- `index.html` : structure du jeu.
- `styles.css` : mise en forme.
- `script.js` : logique de sélection des questions, score et progression.

## Lancer

Ouvrir `index.html` dans un navigateur web pour jouer.
